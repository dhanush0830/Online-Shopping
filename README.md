# Online Shopping Website

A simple AI-assisted online shopping website built using Node.js and Google AI Studio.

This project demonstrates a basic e-commerce platform with modern UI design, product display, and shopping functionality generated and improved using AI-powered development workflows.

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
